# Project Governance

## Overview
This repository serves as an independent, open-source educational portfolio maintained by a Cybersecurity student at Anderson University. It is designed for continuing education, research into Critical Infrastructure Protection (CIP), and as a practical "proof of work" portfolio for future career opportunities in the Operational Technology (OT) defense sector.

## Maintainer Role
*   **Core Maintainer:** The primary author is solely responsible for the technical direction, code merges, and documentation updates of this project.
*   **Independence:** This project is strictly an independent academic and professional development initiative. It is not affiliated with, nor does it represent the views or policies of, Anderson University or any external organizations.

## Community Contributions
While this repository functions primarily as a personal student portfolio, constructive community contributions are welcome. 
*   **Pull Requests (PRs):** PRs targeting code optimization, security hardening, or documentation clarity will be reviewed by the core maintainer. 
*   **Issues:** Users are encouraged to open issues for bug reports or feature requests related to the scripts and compliance files hosted here.
